Using the Visual Flight Path
Your highway in the sky

The Visual Flight Path makes it easier to fly a selected VOR radial or instrument landing system (ILS) approach. The Path puts a "follow me" display in the sky. To stay on course, just follow the display. The Visual Flight Path is especially useful when you're trying to learn the relationship between the VOR needles in your cockpit and the path to the VOR station.

The Visual Flight Path displays a variety of visual shapes for you to follow. You can lock the Path to an ILS or VOR signal, and select the spacing and density of the path that Flight Simulator displays on the window.

Activating the Visual Flight Path

You must have the Visual Flight Path enabled to see the display in the sky.

To activate Visual Flight Path

  1. On the Aircraft menu, click Visual Flight Path.
  2. Select the Enable Visual Flight Path checkbox.
  3. Click OK.

Linking Visual Flight Path to Navigational Aids

You must have a VOR or ILS tuned into one of your radios in order to see the Visual Flight Path when selecting this option.

To lock the Visual Flight Path to a VOR or ILS

  1. Tune a nav radio to the navigation facility you want to track.
  2. In the Visual Flight Path dialog box, select Nav1 or Nav2 in the Linked radio list.
  3. Set any options in the dialog box that you want to change.
  4. Click OK.

Setting Visual Flight Path Shape

Use the Guide Shape list to choose the type of display you want: Rectangles, Telephone poles, Hoops, or Yellow brick road.

To choose how you want the Path to display

  1. In the Guide shape box, select Rectangles, Telephone poles, Yellow brick road, or Hoops.
  2. Click OK.

Setting Visual Flight Path Density

You can choose how densely Flight Simulator places the guides along your path.

To set the Visual Flight Path density

  1. In the Number of guides list, select Low, Medium, or High.
  2. Click OK.

Setting Visual Flight Path Spacing

You can choose how far apart Flight Simulator will place guides along your path.

To set the Visual Flight Path spacing

  1. In the Distance between guides list, select Maximum, Medium, or Minimum.
  2. Click OK.

Plotting an Intercept Course

You can have the Visual Flight Path draw an intercepting path to the radial or final approach course that you have set on the Omni Bearing Selector of a VOR or ILS.

To plot an intercepting path

  1. Select the Plot intercept course checkbox.
  2. Click OK.

Setting Guide Altitude

You can set the altitude at which the guides appear. If the Visual Flight Path is linked to an ILS, the Path will follow the glide slope on the approach phase of the flight.

To set the guide altitude

  1. Type an altitude in the Draw guides at this altitude box.
  2. Click OK.
